Vocabulaire

Pointeur
c’est la souris .
Curseur
c’est | , ou _ clignotant ou autre selon l’application.
Fenêtre
le cadre avec les boutons   –  ☐ ✕ (ou   )
Onglet
souvent en haut, parfois en bas ou sur le côté, qui permet d’avoir plusieurs vues dans une même fenêtre.
Application
ensemble cohérent d’outils qui permet d’effectuer une tâche.

Presse papier
espace mémoire contenant les éléments copiés.

ALT et ALT GR n’ont pas le même comportement ni les mêmes fonctionnalités.

Dans ce document les touches majuscules sont représentées : ⇧
Les touches entourées de deux flèches ↓ALT↓ sont à appuyer et maintenir tout le temps de la manipulation
Les touches à presser simultanément sont séparer par +
Les touches à presser successivement sont séparer par ,

Raccourcis de navigation communs

Excel ne suit pas la même logique pour les déplacements, cf. : chapitre Excel.

  • Début ↖ : déplace le curseur au début de la ligne (Pos1, Début, Home, Origine)
  • Fin FIN : déplace le curseur à la fin de la ligne.
  • △ ou ▽ ou ◁ ou ▷ (= touches directionnelles) : déplace le curseur d’un caractère ou d’une ligne.
  • ⇞ ou ⇟ : déplace le curseur d’une hauteur d’écran, de fenêtre ou de vu (PgUp, PgDn ou Pg AR, Pg AV)
  • CTRL + ◁ ou ▷ : déplace le curseur d’un mot.
  • ⭾ ou F6 : déplace le curseur parmi les éléments de la fenêtre.

Raccourcis de sélection communs

  • CTRL + A : sélectionner tout.
  • ⇧ + [raccourci de navigation] : sélectionne entre le point d’origine et la nouvelle position du curseur.
  • CTRL + [raccourci de navigation] : déplace le curseur sans sélectionner.
  • CTRL + ESPACE : sélectionne/désélectionne l’élément sous le curseur.

Raccourcis de manipulation communs

  • CTRL + Z : annuler la dernière action (sauve des vies depuis 1974)
  • Rétablir la dernière action annulée (selon l’application) :
    • CTRL + Y
    • CTRL + ⇧ + Z
  • CTRL + C : copier (enregistre dans le presse-papier)
  • CTRL + X : couper (enregistre dans le presse-papier, et supprime l’élément sélectionné)
  • CTRL + V : coller (le 1er élément du presse-papier)
  • + V : coller un des éléments de l’historique du presse-papier (À noter que l’on peut épingler un élément pour le coller plus tard, même après redémarrage)
  • CTRL + S : enregistrer.
  • CTRL + P : imprimer (cette page est imprimable 😉)
  • CTRL + ⭾ : passer à l’onglet suivant.
  • CTRL + ⇧ + ⭾ : passer à l’onglet précédent.

Raccourcis de manipulation de texte

Sur une sélection ou sur la position actuelle du curseur.

  • CTRL + U : souligner ( Underline)
  • CTRL + G ou CTRL + B : mettre en gras.
  • CTRL + I : mettre en italique.
  • CTRL + F : Lancer une recherche.
  • CTRL + G ou F3 : dans une recherche va à la prochaine correspondance (+ ⇧ pour aller à la précédente)
  • CTRL + H : Ouvre le menu de remplacement (certaines applications proposent une option nommée Regex ou .*, cf. : https://regex101.com)

Excel

Contrôle au clavier

ALT ou ⇧ + / : déclenche l’affichage des raccourcis des fonctions
les touches saisie permet d’accéder au fonctions, par exemple L permet d’accéder au bandeau “Accueil”, puis V permet d’ouvrir le menu de collage.
Une fois ces chemins appris, il est possible d’aller plus vite en déclenchant la suite de touches : ↓ALT↓ + L, V, V réalise un coller valeur. (ou CTRL + ALT + V, V, ↲)

  • CTRL + ⇧ + + : insérer une cellule, ligne ou colonne.
  • CTRL + - : Supprimer une cellule, ligne ou colonne.
  • ALT + " : Masquer les lignes de la sélection
  • ALT + _ : Afficher les lignes dans la sélection
  • ALT + ( : Masquer les colonnes de la sélection
  • ALT + ) : Afficher les lignes dans la sélection
  • CTRL + L : Insérer un tableau.
  • ⇧ + F11  ou ALT + ⇧ + F1: Insert une nouvelle feuille de calcul.
  • CTRL + ⇧ + ⇞ : passer la feuille active vers la précédente.
  • CTRL + ⇧ + ⇟ : changer la feuille active vers la suivante.
  • CTRL + (F6 ou ⭾) : passer d’un classeur à un autre.
  • CTRL + ⇧ + F1 : passer en plein écran.
  • ALT + ⇧ + ← : Dissocier le groupe de lignes ou de colonnes
  • ALT + ⇧ + → : Grouper les lignes ou les colonnes

Déplacement

  • ⭾ : déplacer le curseur d’une case sur la droite (depuis la dernière cellule d’un tableau, va au début d’une nouvelle ligne).
  • ⇧ + ⭾ : déplacer le curseur d’une case sur la gauche.
  • CTRL + ⭾ : lorsque plusieurs fenêtres d’Excel sont ouvertes, permet de passer de l’une à l’autre.
  • ↲ : déplace le curseur d’une case en bas.

En se déplaçant avec tabulation, la touche ↲ entrée permet d’accéder au début d’une nouvelle ligne de saisie.

  • CTRL + [touche directionnelle] ou Fin, (touche directionnel) : sauter jusqu’à la dernière valeur de la ligne ou de la colonne (!! s’arrête à la première case vide), puis jusqu’à la dernière case de la feuille.

Tous ces déplacements sont utilisables avec ⇧ pour en faire une sélection.

Sélection

  • CTRL + A : tout sélectionner (le tableau, puis la feuille, puis toutes les lignes et colonnes)
  • CTRL + ESPACE : étend la sélection à la/aux colonne/s (de valeur du tableau, puis avec l’entête, puis toute/s la/les colonne/s)
  • ⇧ + ESPACE : étend la sélection à la/aux ligne/s (de valeur, puis avec l’entête, toute/s la/les ligne/s)
  • CTRL + ⇧ + ESPACE : équivalent de CTRL + A
  • ALT + ; : sélectionner uniquement les cellules visibles.
  • (F5, ⭾ * 2, ↲) ou (↓ALT↓ + (L, F, E), ⇧ + 1, L), D, ↲ : sélectionner les cellules vides.

Les sélections discontinus tout au clavier sont possible avec la touche F8, il vous faudra sans doute vous entraîner un peut pour utiliser cette fonction.

  • F8 : passe en mode sélection étendu depuis la cellule active (équivalent au cliqué-glissé ou ⇧ + touche de déplacement) (les modificateurs de déplacement sont utilisables)
  • ⇧ + F8 : Ajoute/supprime une sélection discontinu (permettant de déplacer le curseur en gardant la sélection précédente)
  • ⭾ : permet de déplacer le curseur de cellules en cellules sélectionné et de modifier les sélections précédentes.

Édition de cellules

  • ⇧ + F3 : Ouvre l’insertion de fonction
  • CTRL + ↲ : permet de reproduire la saisie dans toutes les cellules sélectionnées.
  • F2 : ouvrir la saisie rapide de la cellule sous le curseur, puis permet de changer de mode de saisi : Modifier/Entrer.
  • F4 : lors de la saisi de coordonnée de cellule, permet de permuter entre relative, absolu, ligne absolu-colonne relative et ligne relative et colonne absolu (A1 → $A$1 → A$1 → $A1 → A1 ou LC → L1C1 → L1C → LC1 → LC)
  • ALT + ↲  : valide la saisie sans quitter la cellule.
  • CTRL + B : Recopie la valeur de la première cellule sélectionner dans toute la sélection (équivalent du ✚ glisser avec le pointeur).

Tous les raccourcis clavier sur Excel : https://support.microsoft.com/fr-fr/office/raccourcis-clavier-dans-excel-1798d9d5–842a-42b8–9c99–9b7213f0040f

Word

Publipostage

quelques raccourcis permettent une saisi plus aisé des champs de fusions.

  • ALT + F9 : affiche/masque le code des champs de fusions.
  • CTRL + F9 : Insert un champ vide
  • ↓ALT↓ + (S, U, H) : Insertion d’un champ avec l’assistant.
  • ALT + ⇧ + D : Insert la date (mise à jours en continue)
  • ALT + ⇧ + H : Insert l’heure actuel (mise à jour en continue)

Tous les raccourcis clavier sur Word : https://support.microsoft.com/fr-fr/office/raccourcis-clavier-dans-word-95ef89dd-7142–4b50-afb2-f762f663ceb2

Navigateur internet

  • CTRL + clic sur un lien : ouvrir dans un nouvel onglet.
  • CTRL + T : nouvel onglet (new Tab).
  • CTRL + N : Nouvelle fenêtre.
  • CTRL + W : fermer l’onglet actif.
  • CTRL + ⇧ + T : restaure le dernier onglet fermé.
  • CTRL + ⇧ + N : restaure la dernière fenêtre fermé ou ouvre une nouvelle fenêtre privé.
  • CTRL + + : agrandir le texte et les images.
  • CTRL + - : réduire le texte et les images.
  • CTRL + 0 : rétablir la taille du texte et des images par défaut.
  • CTRL + ⭾ : basculer dans l’onglet de droite.
  • CTRL + ⇧ + ⭾ : Basculer dans l’onglet de gauche.
  • CTRL + (1, 2, 3…) : basculer sur l’onglet 1, 2, 3…
  • CTRL + 9 : basculer sur le dernier onglet.
  • F5 ou CTRL + R : recharger la page.
  • CTRL + F5 : supprimer le cache et recharger la page.
  • CTRL + (E ou K) : placer le curseur pour effectuer une recherche.
  • CTRL + ⇧ + B : afficher ou masquer la barre de favoris.

Manipulation des fenêtres et bureaux

  • ↓ALT↓ + ( ⭾ * N ) : naviguer parmi les fenêtres ouvertes de gauche à droite (outil de power-user depuis 1987).
  • ↓ALT↓ + ↓⇧↓ + ( ⭾ * N ) : Naviguer parmi les fenêtres ouvertes de droite à gauche.
  • CTRL + ALT + ⭾ : ouvrir le panneau des fenêtres ouvertes.
  • ALT + ÉCHAP : placer la fenêtre active derrière toutes les autres.
  • + M : minimiser toutes les fenêtres.
  • + D : montrer ou cacher le bureau.
  • ↓↓ + , : montrer le bureau tant que maintenu.
  • + △ ou ▽ ou ◁ ou ▷ : déplacer la fenêtre.
  • + ALT + △ ou ▽ : (Windows 11 uniquement) dimensionner la fenêtre sur la partie supérieur ou inférieur de l’écran.
  • + Z : (Windows 11 uniquement) afficher le menu de positionnement de la fenêtre active, puis entrer le nombre pour sélectionner la disposition, puis l’emplacement.
  • + ⇧ + (◁ ou ▷) : déplacer la fenêtre d’écran en écran.
  • + 1 ou 2 ou 3… : équivaut à un clic sur l’icône correspondant dans la barre d’application (maintenir et appuyer plusieurs fois sur le même nombre si plusieurs fenêtres d’une même application)
  • + CTRL + 1 ou 2 ou 3… : passer d’une fenêtre ouverte à l’autre (correspondant à l’emplacement sur la barre de tâche)
  • + ALT + 1 ou 2 ou 3… : ouvrir le menu (= clic droit) de l’application épinglé au numéro correspondant.
  • (⇧ + clique gauche) ou Clic molette, sur les icônes de la barre d’application : ouvrir une nouvelle fenêtre.
  • Clic molette sur l’aperçu d’une fenêtre de la barre d’application : fermer la fenêtre.
  • CRTL + ⇧ + (clique gauche ou ↲), sur une application dans la barre de tâche, le menu démarrer, le gestionnaire de fichier ou le bureau : Ouvrir l’application en tant qu’administrateur.

Gestion des bureaux virtuels

Les bureaux virtuels peuvent avoir des font d’écran différent un nom personnalisé, une même fenêtre peut être présente sur tous les bureaux. Ces actions n’ont pas de contrôles au clavier, il faut donc les faire à la souris dans le gestionnaire de bureaux (un bouton est par défaut dans la barre de tâches).

  • + ⭾ : Ouvrir le gestionnaire de bureaux.
  • + CTRL + D : Ajouter un bureau virtuel.
  • + CTRL + ◁ ou ▷ : Changer de bureau actif, à gauche ou à droite.
  • + CTRL + F4 : Fermer le bureau virtuel actuel.

Explorateur Windows : Gestionnaire de fichier

  • CTRL + ⇧ + N : créer un nouveau dossier.
  • CTRL + E ou CTRL + F ou F3 : déplacer le curseur dans le champ de recherche.
  • CTRL + T : ouvrir un nouvel onglet (Windows 11)
  • CTRL + D : supprimer le fichier sans mettre à la corbeille.
  • CTRL + W : fermer la fenêtre / l’onglet.
  • CTRL + N : ouvrir une nouvelle fenêtre.
  • ALT + A ou CTRL + L : déplacer le curseur dans le champ chemin.
  • ALT + E : ouvrir les options d’éditions, puis I permet d’inverser la sélection.
  • ALT + O : ouvrir les options des dossiers.
  • ALT + P : activer/désactiver le panneau d’aperçu (P comme Preview)
  • ALT + ⇧ + P : activer/désactiver le panneau de détails.
  • ALT + F : ouvrir le menu déroulant (équivalent clic-droit) plus complet.
  • ALT + G : ouvrir les options d’affichages.
  • ⇧ + clic sur la colonne “Nom” : avec tri actif met les dossiers en haut de la liste.

La recherche peut être faite avec une interprétation littéral, exemple ~*5-a retournera des résultat comportant exactement ‘5-a’; avec des opérateur NOT, OR, >, < ou .. (entre); avec les noms des champs, exemple Nom:abc datedemodification:aujourdhui sorte:video type:mp4 Date:<01/01/2026 taille:10k..1M

Il est possible de lancer une application dans le dossier ouvert en saisissant le nom de l’exécutable dans le champ chemin.
Exemple avec l’invite de commande CMD (ou PowerShell (alias powershell)) : ALT+A (ou CTRL+L) puis cmd (ou powershell) et [↲ ENTREE]

Résoudre un problème

  • ALT : Faire apparaître et déplacer le curseur dans le menu de l’application (Fichier, Édition, Aide…).
  • ALT + espace : ouvrir le menu de la fenêtre (pour la fermer, le redimensionner).
  • ALT + F4 : Fermer l’application, sur le bureau ouvre le menu d’extinction et de redémarrage.
  • CTRL + ⇧ + ÉCHAP : ouvrir le gestionnaire de tâche (si ALT+F4 n’a pas fonctionné)
  • + R : ouvre l’invite de lancement (quelques exemples : taskmgr, cmd, explorer, excel, firefox, regedit, control, ms-settings:, powershell, winget update -r)
  • CTRL + ALT + Suppr : pour verrouillé la session, hanger d’utilisateur, se déconnecter, modifier le mot de passe ou ouvrir le gestionnaire de tâches.
  • + CTRL + ⇧ + B : fait redémarrer l’affichage.

Divers dans le système

  • ⇧ + F10 : ouvre le menu déroulant à l’emplacement du curseur (= clic droit)

  • F2 : renomme l’élément sous le curseur, en cas de sélection multiple : renomme tous éléments en ajoutant un numéro incrémenté selon l’ordre dans lequel ils sont affiché.

  • ÉCHAP : ferme le menu contextuel ouvert, annule la sélection en cours

  • ou CTRL + ÉCHAP : ouvre le menu démarrer, taper des lettres pour chercher une application

  • + ⇧ + S : prendre une capture d’écran

  • + L : Verrouille la session

  • + E : Ouvre le gestionnaire de fichiers

  • + P : ouvre le panneau “Projeter” pour changer l’affichage des écrans en dupliqué, étendu ou éteint

  • ALT + ⇧ + [touche directionnel], dans le menu démarrer : déplace le raccourci sélectionné

Écrire tous les symboles

Avec Windows 10 et 11 un raccourci clavier à été ajouter ouvrant un pop-up :

+ ; (comme pour faire le smiley ;-) 😉) ou . : permettant la saisi des smiley, symboles et caractères spéciaux n’importe où.

Avec la touche ALT maintenue enfoncé on tape le code du caractère sur le pavé numérique puis au relâchement de ALT apparaît le symbole désiré

  • Les principaux

    • É : ↓ALT↓ + 0201
    • È : ↓ALT↓ + 0200
    • Ç : ↓ALT↓ + 0199
    • À : ↓ALT↓ + 0192
    • ↑/↓/→/← : ↓ALT↓ + 24 / 25 / 26 / 27

  • Autres méthodes pour les accents : ^ (circonflexe) et ` (grave)

    • La touche à coté de P → ¨^ (ou ALT GR + 9)
    • La touche ALT GR + 7 è`

  • Pour tous les autres symboles voire cette fiche : https://usefulshortcuts.com/downloads/ALT-Codes.pdf
