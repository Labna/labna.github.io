Mémorandum :
Raccourci clavier

Vocabulaire

  • Pointeur : c’est la souris
  • Curseur : c’est | , ou _ clignotant ou autre selon l’application
  • Fenêtre : le cadre avec les boutons   –   ☐  ✕
  • Onglet : souvent en haut, parfois en bas ou sur le côté, qui permet d’avoir plusieurs vues dans une même fenêtre
  • Application : ensemble cohérent d’outils qui permet d’effectuer une tâche
  • Logiciel : Ensemble d’applications souvent installées sur un poste/PC
  • Serveur : Application qui attend la connexion d’un client pour fournir un service
  • ALT et ALT GR n’ont pas le même comportement ni les mêmes fonctionnalités
  • Presse papier : espace mémoire contenant les éléments copiés

Dans ce document les touches majuscule sont représentées : ⇑
Les touches entourées de deux flèches ↓ALT↓ sont à appuyer et maintenir tout le temps de la manipulation

Raccourcis de navigation communs

Excel ne suit pas la même logique pour les déplacement, CF: chapitre Excel

  1. Début ↖ : déplace le curseur au début de la ligne (Pos1, Début, Home, Origine)
  2. Fin FIN : déplace le curseur à la fin de la ligne
  3. △, ▽, ◁, ▷ (touches directionnelles) : déplace le curseur d’un caractère ou d’une ligne
  4. ⇞, ⇟ : déplace le curseur d’une hauteur d’écran, de fenêtre ou de vu (PgUp, PgDn)
  5. CTRL + (touche directionnelle gauche ou droite) : déplace le curseur d’un mot

Sélection

  • CTRL + A : sélectionner tout
  • ⇑ + [raccourci de navigation] : sélectionne entre le point d’origine et la nouvelle position du curseur
  • CTRL + [raccourci de navigation] : déplace le curseur sans sélectionner
  • CTRL + ESPACE : sélectionne/désélectionne l’élément sous le curseur

Raccourcis de manipulation communs

  • CTRL + Z : annuler la dernière action
  • rétablir la dernière action annulée (dépend de l’application) :
    • CTRL + Y
    • CTRL + ⇑ + Z
  • CTRL + C : copier (enregistre dans le presse papier)
  • CTRL + X : couper (enregistre dans le presse papier, et supprime l’élément sélectionné)
  • CTRL + V : coller (le 1e élément du presse papier)
  • + V : coller un des éléments de l’historique du presse papier (À noter que l’on peut épingler un élément pour le coller plus tard)
  • CTRL + S : enregistrer
  • CTRL + P : imprimer
  • CTRL + ⇆ : passe à l’onglet suivant
  • CTRL + ⇑ + ⇆ : passe à l’onglet précédent

Raccourcis de manipulation de texte

Sur une sélection ou sur la position actuelle du curseur

  • CTRL + U : souligner ( Underline)
  • CTRL + G : mettre en gras
  • CTRL + I : mettre en italique
  • CTRL + F : Lance une recherche
  • CTRL + H : Ouvre le menu de remplacement (certaines applications propose une option nommé Regex ou .*, cf: https://regex101.com/ )

Excel

Déplacement

  • ⇆ (tabulation) : déplace le curseur d’une case sur la droite
  • ⇑ + ⇆ : déplace le curseur d’une case sur la gauche
  • CTRL + ⇆ : lorsque plusieurs fenêtres d’Excel sont ouvertes, permet de passer de l’une à l’autre.
  • ↲ : déplace le curseur d’une case en bas

En se déplaçant avec tabulation, la touche entrée permet d’accéder au début d’une nouvelle ligne de saisie

  • CTRL + (touche directionnelle) : saute jusqu’à la dernière valeur de la ligne ou de la colonne (!! s’arrête à la première case vide), puis jusqu’à la dernière case du document

Tous ces déplacements sont utilisables avec ⇑ pour en faire une sélection

Sélection

  • CTRL + A : tout sélectionner (le tableau, puis la feuille)
  • CTRL + ESPACE : étend la sélection à toute/s la/les colonne/s
  • ⇑ + ESPACE : étend la sélection à toute/s la/les ligne/s
  • CTRL + ⇑ + ESPACE : équivalent de CTRL + A

Édition de cellules

  • F2 : ouvre la saisie rapide de la cellule sous le curseur
  • F4 : lors de la saisi de coordonnée de cellule, permet de permuter entre relative, absolu, ligne absolu-colonne relative et ligne relative et colonne absolu

la bible des raccourcis clavier avec Excel : c’est par ici !

Navigateur internet

  • CTRL + T : nouvel onglet (new tab)
  • CTRL + ⇑ + T : ouvre le dernier onglet fermé
  • CTRL + + : agrandir le texte et les images
  • CTRL + - : réduire le texte et les images
  • CTRL + 0 : rétablit la taille du texte et des images par défault
  • F5 ou CTRL + R : recharge la page
  • CTRL + F5 : supprime le cache et recharge la page

Manipulation des fenêtres et bureaux

  • + ⇆ : Ouvre le gestionnaire de bureaux
  • ↓ALT↓ + ( ⇆ * N ) : navigue parmi les fenêtres ouvertes de gauche à droite
  • ↓ALT↓ + ↓⇑↓ + ( ⇆ * N ) : Navigue parmi les fenêtres ouvertes de droite à gauche
  • ALT + ÉCHAP : place la fenêtre active derrière toutes les autres
  • + M : minimise toutes les fenêtres
  • + D : Montre ou cache le bureau
  • + (touche directionnelle) : déplace la fenêtre
  • + ⇑ + (touche directionnelle droite-gauche) : déplace la fenêtre d’écran en écran
  • + 1 ou 2 ou 3… : équivaut à un clic sur l’icône correspondant dans la barre d’application (maintenir et appuyer plusieurs fois sur le même nombre si plusieurs fenêtres d’une même application)
  • Clic molette sur les icônes de la barre d’application ouvre une nouvelle fenêtre
  • Clic molette sur l’aperçu de la barre d’application d’une fenêtre d’une application ferme la fenêtre

Divers système

  • ⇑ + F10 : ouvre le menu déroulant à l’emplacement du curseur (= clic droit)
  • F2 : renomme l’élément sous le curseur, en cas de sélection multiple : renomme tous éléments en ajoutant un numéro incrémenté selon l’ordre dans lequel ils sont affiché.
  • ÉCHAP : ferme le menu contextuel ouvert, annule la sélection en cours
  • ALT + espace : ouvre le menu de la fenêtre
  • : ouvre le menu démarrer, taper des lettres pour chercher une application
  • + ⇑ + S : prendre une capture d’écran

Écrire tous les symboles

Avec Windows 10 et 11 un raccourci clavier à été ajouter ouvrant un pop-up :

+ ; : comme pour faire le smiley ;-) 😉, permettant la saisi des smiley, symboles et caractères spéciaux n’importe où.

Avec la touche ALT maintenue enfoncé on tape le code du caractère sur le pavé numérique puis au relâchement de ALT apparaît le symbole désiré

  • Les principaux
    • É : ↓ALT↓ + 0201
    • È : ↓ALT↓ + 0200
    • Ç : ↓ALT↓ + 0199
    • À : ↓ALT↓ + 0192
    • ↑/↓/→/← : ↓ALT↓ + 24 / 25 / 26 / 27
  • Autres méthodes pour les accents : ^ (circonflexe) et ` (grave)
    • la touche à coté de P → ¨^ (ou ALT GR + 9)
    • La touche ALT GR + 7 è`
  • Pour tous les autres symboles voire cette fiche : https://usefulshortcuts.com/downloads/ALT-Codes.pdf
