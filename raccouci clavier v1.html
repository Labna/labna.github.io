Mémorandum :
Raccourci clavier

Vocabulaire

  • Pointeur : c’est la souris
  • Curseur : c’est | , ou _ clignotant ou autre selon l’application
  • Fenêtre : le cadre avec les boutons   –   ☐  ✕
  • Onglet : souvent en haut, parfois en bas ou sur le côté, qui permet d’avoir plusieurs vues dans une même fenêtre
  • Application : ensemble cohérent d’outils qui permet d’effectuer une tâche
  • Logiciel : Ensemble d’application souvent installer sur un poste/PC
  • Serveur : Application qui attend la connexion d’un client pour fournir un service
  • ALT et ALT GR n’ont pas le même comportement ni les même fonctionnalités
  • Presse papier : espace mémoire contenant les éléments copiés

Dans ce document les touches majuscule sont représenté : ⇑

Raccourcis de navigation communs

Excel ne suis pas la même logique pour les déplacement, CF: chapitre Excel

  1. Début ↖ : déplace le curseur au début de la ligne (Pos1)
  2. Fin FIN : déplace le curseur à la fin de la ligne
  3. △, ▽, ◁, ▷ (touches directionnelles) : déplace le curseur d’un caractère ou d’une ligne
  4. CTRL + (touche directionnelle gauche ou droite) : déplace le curseur d’un mot

Sélection

  • CTRL + A : sélectionner tout
  • ⇑ + [raccourci de navigation] : sélectionne entre le point d’origine et la nouvelle position du curseur
  • CTRL + [raccourci de navigation] : déplace le curseur sans sélectionner
  • CTRL + ESPACE : sélectionne/désélectionne l’élément sous le curseur

Raccourcis de manipulation communs

  • CTRL + Z : annuler la dernière action
  • rétablir la dernière action annulée (dépend de l’application) :
    • CTRL + Y
    • CTRL + ⇑ + Z
  • CTRL + C : copier (enregistre dans le presse papier)
  • CTRL + X : couper (enregistre dans le presse papier, et supprime l’élément sélectionné)
  • CTRL + V : coller (le 1e élément du presse papier)
  • + V : coller un des éléments de l’historique du presse papier
  • CTRL + S : enregistrer
  • CTRL + P : imprimer

Raccourcis de manipulation de texte

Sur une sélection ou sur la position actuel du curseur

  • CTRL + U : souligner ( Underline)
  • CTRL + G : mettre en gras
  • CTRL + I : mettre en italique

Excel

Déplacement

  • ⇆ (tabulation) : déplace le curseur d’une case sur la droite
  • ⇑ + ⇆ : déplace le curseur d’une case sur la gauche
  • CTRL + ⇆ : lorsque plusieurs fenêtres d’Excel sont ouvertes, permet de passer de l’une à l’autre.
  • ↲ : déplace le curseur d’une case en bas

En se déplaçant avec tabulation, la touche entrée permet d’accéder au début d’une nouvelle ligne de saisie

  • CTRL + (touche directionnelle) : saute jusqu’à la dernière valeur de la ligne ou de la colonne (!! s’arrête à la première case vide), puis jusqu’à la dernière case du document

Tous ces déplacements sont utilisable avec ⇑ pour en faire une sélection

Navigateur internet

  • CTRL + T : nouvel onglet (new tab)
  • CTRL + ⇑ + T : ouvre le dernière onglet fermé
  • CTRL + + : agrandir le texte et les images
  • CTRL + - : réduire le texte et les images
  • CTRL + 0 : rétabli la taille du texte et des images par défault

Manipulation des fenêtres et bureaux

  • + ⇆ : Ouvre le gestionnaire de bureaux
  • ↓ALT↓ + ( ⇆ * N ) : navigue parmi les fenêtres ouvertes de gauche à droite
  • ↓ALT↓ + ↓⇑↓ + ( ⇆ * N ) : Navigue parmi les fenêtres ouvertes de droite à gauche
  • ALT + ÉCHAP : place la fenêtre active derrière toutes les autres
  • + M : minimise toutes les fenêtres
  • + (touche directionnelle) : déplace la fenêtre
  • + ⇑ + (touche directionnelle droite-gauche) : déplace la fenêtre d’écran en écran

Divers système

  • ⇑ + F10 : ouvre le menu déroulant à l’emplacement du curseur (= clic droit)
  • ÉCHAP : ferme le menu contextuel ouvert, annule la sélection en cours
  • ALT + espace : ouvre le menu de la fenêtre
  • : ouvre le menu démarrer, pour chercher une application
  • + ⇑ + S : prendre une capture d’écran
